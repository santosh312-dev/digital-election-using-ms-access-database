# 🗳️ Digital Election System (MS Access)

A simple and efficient **Digital Election System** built using **Microsoft Access**. This project demonstrates how a database-driven application can be used to manage and conduct elections digitally with ease.

## 📌 Features

* 👤 Voter registration and management
* 🗳️ Voting system with secure input
* 📊 Automatic vote counting
* 📋 Candidate management
* 🔒 Basic validation to prevent duplicate voting
* 🖥️ User-friendly MS Access forms

## 🛠️ Technologies Used

* **Microsoft Access (MS Access)**
* Forms for UI
* Tables for data storage
* Queries for data processing
* VBA (if used) for automation

## 📂 Project Structure

* `Tables` → Stores voters, candidates, and votes
* `Forms` → User interface for voting and admin tasks
* `Queries` → Handles vote counting and filtering
* `Modules (if any)` → Contains VBA code

## 🚀 How to Use

1. Download or clone the repository:

   ```bash
   git clone https://github.com/santosh312-dev/digital-election-using-ms-access-database.git
   ```

2. Open the `.accdb` file using **Microsoft Access**

3. Navigate through the forms:

   * Register voters
   * Add candidates
   * Start voting process

4. View results using queries or reports

## ⚠️ Requirements

* Microsoft Access (2016 or later recommended)
* Windows OS

## 🔐 Limitations

* Not suitable for large-scale or national-level elections
* Basic security (no encryption or advanced authentication)
* Single-user or limited multi-user support
